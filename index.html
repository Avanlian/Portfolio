The Legend of Zelda (Unity Remaster)

Game Description

This project is a modern recreation of the original Legend of Zelda. It contains a simple recreation of the first dungeon within the game. You'll be able to experience familair environments, gameplay mechanics and encounters. Explore the Legend of Zelda in a new and enhanced way. Additionally, this project also contains some additions that were not in the original game. This includes a brand new original level, a dash feature, and two brnad new bosses to give you a challenge. Enjoy!'

Technologies Used

This project was made through the use of the Unity game engine.
