The Legend of Zelda (Unity Remaster)

Game Description

This project is a modern recreation of the original Legend of Zelda. It contains a simple recreation of the first dungeon within the game. You'll be able to experience familair environments, gameplay mechanics and encounters. Explore the Legend of Zelda in a new and enhanced way. Additionally, this project also contains some additions that were not in the original game. This includes a brand new original level, a dash feature, and two brnad new bosses to give you a challenge(Press 4 when you want to see the new level). Enjoy!'

Technologies Used

This project was made through the use of the Unity game engine.

Contributions

Mechanics within this project were split up relatively evenly. My additions include:

- Bomb Weapons

- Boomerang Weapons

- The Camera Controller

- The Goriya Enemies

- Doors and room transitions

- An original new boss

- And many more

Wizard Wars

Game Description

This project is a short single player action game. The player will go through a short tutorial in which they learn the mechanics of the game. After doing so, the player will enter a room and fight a set of 3 bosses in order to win.
